# Gemini Model Overview for ADK

ADK supports multiple Gemini models with varying capabilities, performance characteristics, and pricing. Selecting the right model involves balancing reasoning strength, latency, multimodal support, and cost based on your application’s needs.

---

## Model Capabilities

| Model | Description | Input Types | Best For |
|------|------------|------------|----------|
| gemini-2.5-pro | Most powerful reasoning model with maximum accuracy | Audio, images, video, text | Complex coding, deep reasoning, advanced multimodal understanding |
| gemini-2.5-flash | Best price-to-performance balance | Audio, images, video, text | High-throughput tasks requiring reasoning with low latency |
| gemini-2.0-flash | Newest multimodal model with improved agentic capabilities | Audio, images, video, text | Low latency, enhanced performance, agent workflows |
| gemini-2.0-flash-lite | Optimized for speed and efficiency | Audio, images, video, text | Cost-sensitive, low-latency applications |
| gemini-1.5-flash | Versatile and reliable general-purpose model | Audio, images, video, text | Fast, broad task coverage |
| gemini-1.5-flash-8b | Lightweight and fast model | Audio, images, video, text | High-volume, lower-intelligence workloads |
| gemini-1.5-pro | Strong reasoning and analysis capabilities | Audio, images, video, text | Complex reasoning with slightly lower cost than 2.5-pro |

## Token Information

- One token is approximately 4 characters
- 100 tokens correspond to roughly 60–80 English words
- Pricing is calculated separately for:
  - **Input tokens** (prompts sent to the model)
  - **Output tokens** (responses generated by the model)

---

## Model Selection Guidelines

1. **Budget-Conscious Applications**  
   Start with `gemini-2.0-flash-lite`.

2. **Balanced Performance and Cost**  
   Use `gemini-2.0-flash` or `gemini-2.5-flash`.

3. **Advanced Reasoning and Accuracy**  
   Choose `gemini-2.5-pro`.

4. **Production Deployments**  
   Prefer stable, non-preview models for reliability.
